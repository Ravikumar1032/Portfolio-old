## 🏗️ Installation & Setup
1. **Clone the repository**
   ```bash
   git clone https://github.com/Ravikumar1032/Portfolio.git
   cd Portfolio
   ```
2. **Install dependencies**
    ```bash
    npm install
    ```
3. **Run the app**
    ```bash
    npm start
    ```
Open http://localhost:3000 in your browser.
